# databattle2026--meteorage
Développer un modèle probabiliste capable d’estimer la fin réelle d’un orage, en analysant la dynamique spatio-temporelle des éclairs dans un rayon de 30 km autour de plusieurs aéroports européens. Identifier et analyser les spécificités de chaque lieu et les principaux types d’orage existant dans les données.
